## Getting Started

1. Copy `.env.example` to `.env.local` and add your Supabase project URL and anon key.
2. Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the starter. Update `src/app/page.tsx` to point the Supabase query at your own table or RPC call. The sample query reads a few rows from a `projects` table and shows any errors returned by Supabase.
